Acerca de

Este proyecto fue desarrollado como parte de la clase de Desarrollo Web en Entorno Cliente en la Facultad de Ingeniería de la Universidad de Managua (UdeM).

El objetivo principal del proyecto es aplicar los conocimientos adquiridos durante el curso para crear una aplicación web funcional y efectiva. A lo largo del desarrollo, se utilizaron diversas tecnologías y prácticas recomendadas en el desarrollo web moderno, incluyendo HTML, CSS y JavaScript.

Además de cumplir con los requisitos académicos, este proyecto busca proporcionar una base sólida para futuros desarrollos y servir como ejemplo práctico de las habilidades adquiridas en el curso.
